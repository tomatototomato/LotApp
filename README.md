# ①課題番号１　課題：じゃんけんアプリ

作品名：スクラッチくじ シュミレーションツール

## ②課題内容（どんな作品か）

- スクラッチくじの簡易シュミレーションツールです。
  単純にランダムに数字を生成しているだけなので、本家のロジックとはおそらく異なるかと思います。

- 1〜7までの数字に任意の出現確率を設定することで、
  javascript側で生成した乱数から出現確率に準じた数字がセットされ、
  同じ数字が５つ以上出現すると該当するあたりが表示される仕組みです。
